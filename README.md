# NSObjectExtend
`NSObjectExtend` Category for `UIKit` and `CoreLocation`. Extend very useful method.


# Quick start

NSObjectExtend is available on [CocoaPods](http://cocoapods.org).  Add the following to your Podfile:

```ruby
pod 'NSObjectExtend'
```
